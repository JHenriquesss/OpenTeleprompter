================================
OpenPrompter RS — Portable Version
================================

Thank you for downloading OpenPrompter RS!

This is the portable (no-install) version. Just unzip and run.

HOW TO USE:
1. Extract all files to any folder (USB drive, desktop, etc.).
2. Double-click "OpenPrompter RS.exe" to launch.
3. Your scripts and settings are saved in the same folder.

SYSTEM REQUIREMENTS:
- Windows 10 or newer (64-bit)
- Microsoft Edge WebView2 Runtime
  (pre-installed on Windows 11; on Windows 10 it may install automatically)

ABOUT THE SMARTSCREEN WARNING:
Windows may show "Windows protected your PC" the first time you run this app.
This is because the executable is unsigned (no paid certificate).
It does NOT mean the app is unsafe.
Click "More info" then "Run anyway" to proceed.
